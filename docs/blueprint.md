# **App Name**: Delhi Estate Luxe

## Core Features:

- Luxury Property Listings: Showcase high-end properties in Delhi with detailed information and high-quality images.
- Advanced Property Filters: Enable users to filter properties by price, locality, type, and other criteria.
- Builder Tie-up Showcase: Display builder partnerships with dedicated sections and featured projects.
- Interior Design Portfolio: Exhibit interior design projects with galleries and descriptions to attract potential clients.
- AI Property Recommendation Tool: Employ an AI-driven tool to recommend properties based on user preferences and search history.
- Integrated Contact Forms and Chat: Include contact forms and direct WhatsApp/call buttons for easy communication.
- SEO Optimization: Optimize website content and structure for target keywords like 'Delhi properties' and 'luxury homes Delhi'.

## Style Guidelines:

- Primary color: Midnight Blue (#191970) to evoke trust and luxury.
- Background color: Dark Gray (#292929), offering a sophisticated backdrop for the content.
- Accent color: Gold (#D4AF37) to highlight important elements and CTAs, adding a touch of elegance.
- Use White (#FFFFFF) for text and highlights on dark backgrounds to ensure readability and contrast.
- Body and headline font: 'Poppins' (sans-serif) for a clean and modern aesthetic.
- Phone-first responsive design with a fixed bottom navigation bar on mobile and a top navigation bar on desktop/tablet.
- Smooth micro-animations to enhance user experience and add subtle sophistication.
- Minimalist, luxury-style icons to complement the overall modern design.
